# Projeto #2: Árvore de links (HTML/CSS/JS) - Repositório acadêmico voltado ao aprendizado.
Este é o meu terceiro projeto prático focado nos fundamentos do desenvolvimento web front-end. O objetivo foi aplicar conceitos essenciais de estruturação e estilização, criando uma landing page/árvore de links, funcional, limpo e bem organizado.

# 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação do conteúdo.
- **CSS3**: Estilização, layout e responsividade básica.
